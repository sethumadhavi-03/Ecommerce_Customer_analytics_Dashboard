#                                                                      🛒 E-Commerce Customer Analytics Dashboard

## 📌 Project Overview

This project analyzes an E-Commerce dataset using SQL and Power BI to uncover business insights, customer behavior patterns, sales trends, and revenue drivers.
The objective was to transform raw transactional data into an interactive dashboard that supports data-driven decision-making.

---

## 🎯 Business Questions Answered

### Sales Analysis

* Total Revenue Generated
* Total Orders Placed
* Average Order Value
* Monthly Revenue Trends

### Customer Analysis

* Total Customers
* Customer Spending Patterns
* Top Customer Cities

### Product Analysis

* Top Product Categories by Revenue
* Revenue Contribution by Category

### Seller Analysis

* Top Sellers by Revenue
* Seller Performance Comparison

### Geographic Analysis

* Revenue by State
* State-wise Sales Distribution

### Payment Analysis

* Orders by Payment Type
